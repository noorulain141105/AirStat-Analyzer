# AirStat Analyzer
# Spring 2026
## Probability & Statistics Project (MT2005)

AirStat Analyzer is a Flask-based web application developed for the Probability & Statistics course. The application allows users to analyze air quality datasets using statistical techniques, probability distributions, regression analysis, and interactive visualizations.

---

## Features

✅ Upload Custom CSV Dataset

✅ Built-in Demo Dataset

✅ Descriptive Statistics
- Mean
- Median
- Mode
- Variance
- Standard Deviation

✅ Confidence Intervals (95%)

✅ Histograms

✅ Boxplots

✅ Scatter Plot Analysis

✅ Correlation Heatmap

✅ Normal Distribution Fitting

✅ Binomial Distribution

✅ Poisson Distribution

✅ Multiple Linear Regression

✅ AQI Prediction

✅ R² Score & RMSE Evaluation

## Installation

### Clone Repository

```bash

```

```bash

```

### Run Application

```bash
python app.py
```

### Open Browser

```text
http://127.0.0.1:5000
```
